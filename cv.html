FreeGamesForYourWebsite is a website for game publishers. Using this website the publisher can add games to their site with one click either by download or embed code. They can also use xml and json feeds to get game information and files for their selected criteria. The site also boasts a list of 1000+ games, which are free to download for publishers.

The Ad Server was created so that our Ad Team could focus on getting more clients. With the Ad Server, advertisers can create their campaigns, add new lineitems and create new creatives. The Ad Server system communicates with the Google Double Click For Publishers API. After the creation of a valid campaign with associated creatives, the lineitem is uploaded to DFP. Once the loneitem starts delivering, money is deducted from the users account based on the amount of impressions delivered.
